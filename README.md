# CAPTCHA: Am I Human?

## Identity Horror Game

A web-based psychological horror game disguised as a CAPTCHA verification system.

### Concept
You're taking CAPTCHAs to prove you're human. Normal at first: select traffic lights, crosswalks. Then it gets personal. Then it gets terrifying.

### Gameplay Progression
1. **Normal** - Standard CAPTCHA tasks (traffic lights, storefronts, bicycles)
2. **Uncanny** - Tasks that feel slightly off ("Select all images that feel wrong")
3. **Personal** - It knows things about you ("Select all images containing YOUR face")
4. **Invasive** - It asks impossible questions ("Click where you were last night")
5. **Horror** - The revelation ("Type what you're afraid of")

### Technical Stack
- Pure HTML/CSS/JavaScript (no framework needed)
- No backend required for MVP
- Deployable anywhere (GitHub Pages, Netlify, itch.io)
